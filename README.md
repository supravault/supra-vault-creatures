# Vault Creatures Bot

A Telegram-based creature collecting and battle game built with Node.js and Telegraf.  
Players discover creatures, battle in turn-based encounters, and level up their pets using Ascend Items.

---

## Features

- Wild encounters with randomized creatures  
- Turn-based combat (player turn → enemy turn → player turn)  
- Founder pet (Grimnex) with a unique 4-move set  
- Non-founder pets always have:
  - Strike  
  - Guard Up  
  - One random special move (burn, bleed, freeze, shock, blind, regen)  
- Ascend Item system to level up pets  
- Item gifting between players  
- Simple JSON storage
